# Sidechain API

Upload, Get works, like, comment, get work, get stems

**Data types**

Contract Address : String

Wallet Address : String

File Link : String

Sidechain Metadata : {

data : [

{

description : String

name : String

image : String (link to file)

asset_specific_data : {

audio : File Link

tags : [String]

stems : [

{track_name  : String, file : File Link}

]

creator : String | Wallet Address

comments : [{commenter : Wallet Address, text : String, time : int}]

likes : [{liker : Wallet Address, time : int}]

timestamp : int

}

}

]

}
